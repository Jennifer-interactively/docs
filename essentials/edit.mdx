## Add steps

Add content by importing steps from your library, uploading files from your computer, or inserting a chapter. 

### Import steps from another demo (Library)

1. Click `Add step` or the `+` button between steps
2. Select **Upload from Library**
3. Click the `+` button on the image or video you want to add to the demo
4. Add multiple steps by selecting them and clicking `Add to demo`

<Info>
  All recordings are automatically saved to your media library
</Info>

### Import files from computer

1. Click `Add step` or the `+` button between steps
2. Select **Upload from Computer**
3. Add files(s)

### Add chapter

1. Click `Add step` or the `+` button between steps
2. Select **Add Chapter **
3. Set the **Theme**: Light or dark theme
4. Add a **Title** and **Subtitle**
5. **Configure button settings: **Edit text and select colors

## Duplicate steps

To duplicate:

1. Select the three-dot button `...`
2. Select **Duplicate** 

## Delete steps

To delete:

1. Select the three-dot button `...`
2. Select **Delete**

Once you've recorded your demo and its been uploaded, select the `Edit` tab to open the demo editor.
