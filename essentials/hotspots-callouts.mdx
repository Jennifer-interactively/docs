## Font

Select your demos font using the dropdown menu (default font is Inter).

<Info>
  All text elements in the demo will use the selected font
</Info>

## Colors

Set colors for hotspots, callouts, and buttons using the color picker or HEX codes.

<Tip>
  Set a theme to apply consistent styling across all your demos.
</Tip>

## Watermark

Include a watermark in the lower right corner of your demos.

### Disable watermark

1. Toggle the watermark off by clicking the switch

### Add a custom watermark

1. Toggle the watermark on
2. Click Upload
3. Choose your logo file from your computer
4. Enter your website URL in the text field

## Dark mode

Dark mode changes the background and UI of your demo to a dark color palette, making it ideal for showcasing dark-themed products and websites.

- **Toggle between Light and Dark mode** by clicking the moon or sun icon in the editor toolbar. The icon switches based on the current mode.
