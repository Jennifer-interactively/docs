## Font

Select the font for your demo using the dropdown menu (default is Inter). 

<Info>
  All text elements in the demo will use the selected font
</Info>

## Colors

Define colors for hotspots, callouts, and buttons using the color picker or HEX codes.

<Tip>
  Set a theme to ensure consistent styling across all your demos.
</Tip>
